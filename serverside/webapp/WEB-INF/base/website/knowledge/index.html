Getting Started

eMediaLibrary Entities

Entities enables files to be organized in a more automated way, it also makes for faster searching! Entity based systems are a more natural way to store your files than just folders on your computer. When you add facial and object recognition plus automated tags it's even more exciting.

Image Editor

eMediaLibrary has a built in image editor so you don’t have to swap between apps to do simple edits.
Crop, rotate, add filters, text, and convert in the image editor for a more fluid workflow.

Facial Recognition

When you upload a folder, or a collection of files to eMediaLibary, you have the option to enable facial recognition. Once facial recognition is enabled, the software will start recognizing faces and grouping them into facial profiles. The more you use it, the more it will learn, making it more effective over time! You can use this feature for all sorts of useful applications.

High Powered Searches

Search in 3 different ways with eMediaLibrary. Use the top search bar to browse files by keyword. You can also manually browse entities and directories. To make your process even easier, the summary sidebar allows the user to narrow down and filter results even more by size, date, type, face, and much more!

Organize your files with Tags and Metadata

eMediaLibrary has an amazing system of tags, profiles, entities and metadata to help you find what you are looking for and organize your files. Metadata is created when a user enters details on the files they are uploading, and it’s also created automatically with artificial intelligence.
